# Prompt Kit

专业的 Prompt 知识库和工作流工具

## 功能特性

### ✅ 已实现的核心功能

1. **Prompt 管理**
   - 创建、编辑、删除 Prompt
   - 结构化字段管理（角色、目标、输入槽位、输出格式、约束、示例）
   - 标签与分类管理
   - 版本管理与变更日志

2. **收藏与导入**
   - 从剪贴板快速粘贴并自动解析结构化字段
   - 批量导入（JSON/CSV 格式）
   - 一键导出（JSON/CSV/Markdown）

3. **搜索与过滤**
   - 全文搜索
   - 标签过滤
   - 评分过滤
   - 可见性过滤

4. **运行与复用**
   - Prompt 运行界面
   - 变量填充
   - 多模型选择（模拟）
   - 运行记录保存

5. **看板与统计**
   - 总 Prompt 数、收藏数、运行次数统计
   - 最常用 Prompt 展示
   - 最高评分 Prompt 展示

6. **数据存储**
   - IndexedDB 本地存储
   - 数据持久化
   - 导入导出功能

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **路由**: React Router
- **数据存储**: IndexedDB (原生 API)
- **样式**: CSS3

## 核心功能说明

### Prompt 数据结构

每个 Prompt 包含以下核心字段：

- **基本信息**: 标题、描述、语言、来源
- **结构化模板**: 角色、目标、输入槽位、输出格式、风格、约束、示例
- **运行配置**: 目标模型、参数预设、上下文拼接策略
- **组织属性**: 标签、分类、项目归属、优先级、场景
- **质量与评估**: 评分、使用次数、成功率、评审记录
- **版本与协作**: 版本号、变更说明、审批状态、可见范围

### 输入槽位（占位符）

支持多种类型的输入槽位：
- 文本 (text)
- 多行文本 (multiline)
- 数字 (number)
- 布尔值 (boolean)
- 选择 (select)

每个槽位可以配置：
- 名称
- 类型
- 默认值
- 验证规则
- 提示信息
- 是否必填

### 导入导出格式

- **JSON**: 完整数据结构，包含所有字段
- **CSV**: 常用字段的表格格式
- **Markdown**: 文档化格式，便于阅读和分享

## 使用指南

### 创建 Prompt

1. 点击侧边栏的"新建 Prompt"按钮
2. 填写基本信息（标题、描述等）
3. 填写结构化模板字段
4. 添加输入槽位（如果需要）
5. 设置组织属性（标签、分类等）
6. 点击"保存"

### 快速粘贴解析

1. 在编辑页面，粘贴 Prompt 文本到"快速粘贴解析"区域
2. 点击"解析并创建"按钮
3. 系统会自动提取结构化字段

### 运行 Prompt

1. 在 Prompt 列表页面，点击"运行"按钮
2. 填写输入参数
3. 选择模型
4. 点击"运行"按钮
5. 查看运行结果

### 导入导出

**导出**:
1. 在 Prompt 列表页面，点击"导出"按钮
2. 选择导出格式（JSON/CSV/Markdown）
3. 文件会自动下载

**导入**:
1. 在 Prompt 列表页面，点击"导入"按钮
2. 选择 JSON 或 CSV 文件
3. 系统会自动导入并保存
